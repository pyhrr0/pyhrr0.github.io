Datastructures: Merkle tree

Summary Merkle trees (aka hash trees) are tree-like datastructures, that separate data validation from the the data itself. These trees consist of the following items: Leaves (aka outer nodes aka leaf node) Data objects, that are labelled with: hash(associated_data).
